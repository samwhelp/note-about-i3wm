## 如何安裝我的設定檔

執行下面指令，安裝我的設定檔到「~/.config/i3/config」。

``` sh
curl -fLo ~/.config/i3/config --create-dirs \
	'https://raw.githubusercontent.com/samwhelp/note-about-i3wm/gh-pages/_demo/config/i3wm-config/main/config/i3/config'
```
